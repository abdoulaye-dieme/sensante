# SenSante

Assistant de pré-diagnostic médical pour le Sénégal.

## Description
SenSante utilise le Machine Learning pour aider au pré-diagnostic
des maladies comme le paludisme, la grippe et la typhoïde.

## Structure du projet

- data/ : données patients
- models/ : modèle ML
- api/ : API
- frontend/ : interface web
- notebooks/ : scripts d'exploration
